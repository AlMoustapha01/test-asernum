## Getting Started

First install dependencies

```bash 
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Or simply run:

```bash
docker compose up -d
```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
